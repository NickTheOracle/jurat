# Jurat (prototype)

Static prototype for a legal intake → USCIS form automation workflow.

## Run

Open `public/index.html` in a browser.

## What's included

- Assistant intake flow (select form → intake → generate PDF)
- Client intake link (simple URL for clients)
- Client list with N-400 PDF download
- Form review panel for N-400
- Official USCIS N-400 PDF auto-fill (client-side, uppercase)

This is a static UI prototype with browser-only storage and a real N-400 PDF template.

## Pages

- Assistant: `public/index.html`
- Client intake: `public/client.html`
